# Intro to Version Control: Lesson Plan

The course is delivered over 2 days with 3 hours per day, 6 hours in total. The plan below assumes a scheduled time of 10:00 to 13:00.

## Links 

For quick copy-pasting into chat:

- Workshop info:  https://coding-for-reproducible-research.github.io/CfRR_Courses/programme_information/intro_to_version_control.html 
- Self study notes:   https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/section_landing_pages/introduction_to_version_control.html


## Day 1

### Preparation

- Helpers 
    - Have links ready for copy-paste: Registration link and Link to first lesson
    - Have Git bash or similar installed
- Leader:
    - Open Powerpoint from Anne
    - Open browser with lecture material at start point
    - Start Git bash

### Lesson plan

- 10:00 **Introduction**
    - Show by starting 4 slides (supplied by Anne). 
        - 1st slide:  Introduce self and helpers, Ask participants to complete registration
        - 2nd slide: Introduce Code of conduct (students to read for themselves)
        - 3rd slide: Point out programme funding (students to read)
        - 4th slide: Mention feedback will be sought next week
    - Ask participants to leave cameras on
    - Ask whether it's OK to record session, start recording if OK
    - Introduce format of the session
        - 3 segments, 2 breaks of 15 minutes
        - breaks are at 10:45 -> 11:00 and 11:50 -> 12:05
        - 12:50 end of lecture, time for questions until 13:00
    - Copy-paste links into the chat, ask students to follow along
    - Mention the quiz at the bottom of every page, results are not recorded

- 10:15 **Segment 1** Start presentation material
    - [Motivation: Why use Version Control Systems?](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/motivation.html)
    - [Git vs GitHub](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/git_vs_github.html)
    - [Setting up Git and GitHub](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/configuring_git.html) (Part 1)
        - Setting up Git


- 10:45 - 11:00: **First break**
    - Helpers: use breakout rooms to solve installation/command-shell issues
    - Leader: rest voice and get coffee

- 11:00 **Segment 2** Continue lecture material
    - [Setting up Git and GitHub](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/configuring_git.html) (Part 2)
        - Set up Github credentials
    - [Making Repositories](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/making_repos.html)
    - [An Example Repository](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/example_repo.html)
    - [Recording Changes](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/recording_changes.html)

- 11:50 - 12:05: **Second break**

- 12:05 - 12:50: **Segment 3** Continue lecture material
    - [Viewing History and Changes](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/history_and_changes.html)
    - [Recording Changes â€“ Revisited](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/recording_changes_revisited.html)
    - [Ignoring Files](https://coding-for-reproducible-research.github.io/CfRR_Courses/individual_modules/introduction_to_version_control/ignoring_files.html)

- 12:50: **End of formal material**
    - Time for questions, discussion, troubleshooting

- 13:00: **Close of session**
